How It Works

The ReAct pattern — foundation of every production AI agent

💭
THINK
LLM reasons about what the user needs and which tool (if any) to use
→
⚡
ACT
Agent calls a tool with parameters, or produces a final answer
→
👁️
OBSERVE
Tool result is fed back to the LLM as new context
↩ loop

The ReAct Loop in Code

for iteration in range(MAX_ITERATIONS):
    # 1. THINK — ask the LLM
    response = llm.generate(prompt + memory + tools)
    parsed   = parse(response)  # extract thought/action/answer

    # 2. ANSWER — if LLM has what it needs
    if parsed.final_answer:
        yield AgentStep("answer", parsed.final_answer)
        return

    # 3. ACT — call the tool
    observation = run_tool(parsed.action, parsed.action_input)

    # 4. OBSERVE — add result to memory and loop again
    memory.add("tool", observation)
    continue  # ← back to top, now LLM has more context
